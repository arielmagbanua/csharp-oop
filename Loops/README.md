# Loops

A loop is used to repeat a block of code until the specified condition is met.

![loops](loops.png|width=100px)
<img src='loops.png' width='100'>

## For Loop

```csharp
for (int i = 0; i < 5; i++)
{
    Console.Write(i + " ");
}
```
```shell
# output
0 1 2 3 4 
```

## While Loop

```csharp
int num = 0;
while (num < 5) {
    Console.Write(num + " ");
    num++;
}
```
```shell
# output
0 1 2 3 4 
```
